# 📦 文献导读助手 2.0 - 分享包

## 🎯 技能简介

**文献导读助手 2.0** 是一个全自动的学术文献导读技能，一键完成「PDF 提取→文献导读」全流程。

**核心功能**：
- ✅ 自动 PDF 提取（使用 MinerU CLI）
- ✅ 4 层递进式导读（直觉层→概念层→技术层→批判层）
- ✅ LaTeX 公式自动渲染
- ✅ 智能模式选择（flash-extract vs extract）
- ✅ 一键保存到 `./文献导读/` 目录

## 🚀 快速开始（给测试者的说明）

### 前提条件

1. **OpenClaw / LobsterAI**：确保已安装并运行
2. **Node.js**：用于安装 MinerU CLI（v16+）

### 步骤 1：安装技能

将 `academic-literature-guide-v2` 文件夹复制到 LobsterAI 技能目录：

**Windows**：
```powershell
# 方法 1：手动复制
# 将文件夹复制到：
C:\Users\<你的用户名>\AppData\Roaming\LobsterAI\SKILLs\

# 方法 2：命令行
Copy-Item -Path ".\academic-literature-guide-v2" -Destination "$env:APPDATA\LobsterAI\SKILLs\" -Recurse
```

**macOS / Linux**：
```bash
cp -r academic-literature-guide-v2 ~/.lobsterai/SKILLs/
# 或
cp -r academic-literature-guide-v2 ~/Library/Application Support/LobsterAI/SKILLs/
```

### 步骤 2：安装 MinerU CLI（必需）

```bash
# 通过 npm 安装（推荐）
npm install -g mineru-open-api

# 验证安装
mineru-open-api version
```

应看到类似输出：
```
mineru-open-api version v0.2.1
  commit: faf8069a
  built:  2026-03-20T10:44:12Z
```

### 步骤 3：配置 API Token（可选，用于大文件）

**获取 Token**：
1. 访问 https://mineru.net/apiManage/token
2. 创建免费账号
3. 复制 Token

**配置 Token**：
```bash
mineru-open-api auth
```

按提示输入 Token。

### 步骤 4：重启 LobsterAI（如需要）

如果技能未立即生效，重启 LobsterAI：
```bash
# Windows（PowerShell）
Restart-Service -Name LobsterAI

# 或手动重启 LobsterAI 应用
```

## 📖 使用方法

### 基本用法

在 LobsterAI 中输入以下任一命令：

**提供 PDF 文件路径**：
```
帮我解读这篇 PDF：C:\Users\...\论文.pdf
```

**提供文件名（自动搜索）**：
```
解读一下"烧钱行为"那篇 PDF
```

**提供 URL 链接**：
```
帮我解读这个链接的论文：https://arxiv.org/pdf/2509.22186
```

**提供 DOI 或标题**：
```
帮我生成 10.1080/xxxx 这篇论文的导读
```

### 完整工作流程

1. 用户提供 PDF/链接
2. 技能自动询问读者背景（科研小白/进阶人员/资深学者）
3. 自动调用 MinerU 提取 PDF
   - 小文件（<10MB, <20 页）→ flash-extract（快速）
   - 大文件 → extract（精确，需要 Token）
4. 基于提取内容生成 4 层导读报告
5. 自动保存到 `./文献导读/` 目录

## 🔍 技能特性详解

### 1. 智能模式选择

| 条件 | 使用模式 | 说明 |
|------|---------|------|
| 文件 < 10 MB AND < 20 页 | flash-extract | 快速，无需 Token |
| 文件 ≥ 10 MB OR ≥ 20 页 | extract | 精确，需要 Token |
| 用户明确要求表格/公式识别 | extract | 精确模式支持表格/公式 |

### 2. 分级导读

根据用户背景自动调整：

| 读者层级 | 术语密度 | Layer 1-2 篇幅 | Layer 3-4 篇幅 |
|----------|----------|---------------|---------------|
| 科研小白 | ≤5 术语/千字 | 50-60% | 20-30% |
| 进阶人员 | ≤15 术语/千字 | 30-40% | 40-50% |
| 资深学者 | 不限 | 10-20% | 60-70% |

### 3. LaTeX 公式渲染

- 行内公式：`$...$`
- 独立公式块：`$$...$$`
- 多行公式：`$$\begin{aligned}...\end{aligned}$$`
- 禁止 Unicode 数学符号（𝑥², 𝛼, 𝛽）

## 🐛 故障排查

### MinerU 未安装

**错误**：
```
mineru-open-api：无法识别为 cmdlet、函数、脚本文件
```

**解决**：
```bash
npm install -g mineru-open-api
```

### Token 未配置

**错误**：
```
API token required for precision extraction
```

**解决**：
```bash
mineru-open-api auth
```

### 提取失败

**可能原因**：
1. 文件加密或损坏
2. MinerU API 配额用尽
3. 网络连接问题

**解决**：
- 检查文件是否可以正常打开
- 等待几分钟后重试
- 检查 Token 有效性：`mineru-open-api auth --verify`
