## 回复策略指南

### 回复原则

- [ ] 逐条回复，不遗漏
- [ ] 先感谢，再回应
- [ ] 态度诚恳，不争辩
- [ ] 具体说明修改内容
- [ ] 明确标注修改位置

### 不同意见的处理

| 审稿意见类型 | 回复策略 | 示例 |
|-------------|---------|------|
| **完全同意** | 感谢 + 说明修改 | "感谢指出，已按建议修改..." |
| **部分同意** | 承认合理部分 + 解释 | "感谢建议。关于 X 部分已修改；关于 Y 部分..." |
| **不同意** | 礼貌解释 + 补充论证 | "感谢审稿人的思考。我们考虑了这一观点，但认为...理由如下..." |
| **误解** | 澄清 + 补充说明 | "感谢提问。可能是表述不清所致，我们已补充说明..." |

### 常见审稿意见回复模板

**意见：内生性问题未处理**
```
感谢审稿专家的宝贵意见。针对内生性问题，我们做了如下补充：

1. 工具变量法：我们采用 [工具变量名称] 作为工具变量，第一阶段 F 统计量为 XX（>10），
   表明不存在弱工具变量问题。2SLS 回归结果见表 X（第 X 列），核心系数方向和显著性
   与基准结果一致。

2. 稳健性检验：我们还进行了以下检验：
   - [检验 1 说明]
   - [检验 2 说明]
   
上述结果表明，我们的结论在控制内生性后依然稳健。
```

**意见：稳健性检验不充分**
```
感谢审稿专家的建议。我们补充了以下稳健性检验（见表 X-Y）：

1. 替换变量度量：使用 [替代度量] 替代原变量，结果不变。
2. 改变样本范围：剔除 [特殊样本] 后，结果不变。
3. 改变聚类层级：在 [行业/省份] 层面聚类，结果不变。
4. 安慰剂检验：随机生成处理组，重复 1000 次，真实估计值位于分布尾部。

上述检验表明，我们的结论是稳健的。
```
