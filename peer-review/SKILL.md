# 同行评议专家

## 核心能力

- **双轨评审模式**：正常版用于常规场景，批判版用于深度场景
- **五大批判性原则**：因果推断边界、未测量变量威胁、可推广性边界、学术辩论定位、缺失关键文献
- **人文社科方法论**：因果识别策略审查、内生性处理、稳健性检验、机制检验

---

## 双轨评审模式

| 场景 | 推荐模式 |
|------|---------|
| 硕士学位论文评审 | 双轨模式 |
| 期刊初审 | 正常版 |
| 期刊外审 | 批判版 |
| 博士论文评审 | 批判版 |
| 基金申请书评审 | 批判版 |
| 批量评审（>5 篇） | 正常版 |

**批判版框架**：
```
一、执行摘要（200 字）
二、因果推断审计（原则 1+2）
三、可推广性审计（原则 3）
四、学术辩论定位（原则 4+5）
五、传统维度速评
六、修改优先级与最终结论
```

---

## 五大批判性原则

### 原则 1：因果推断边界
**能合理推出什么 vs 不能推出什么**

| 能合理推出 | 不能合理推出 |
|-----------|-------------|
| 相关性发现 | 因果效应断言 |
| 时间序列特征 | 规范性判断 |

### 原则 2：未测量变量威胁
**关键变量是否被遗漏？威胁等级？**

| 未测量变量 | 潜在影响 | 威胁等级 |
|-----------|---------|---------|
| 平台实际使用情况 | 高 | 🔴极高 |

### 原则 3：可推广性边界
**结论能推广多远？**

| 推广目标 | 可推广性 | 理由 |
|---------|---------|------|
| 中国非上市公司 | 低 | 治理较弱 |
| 发达国家 | 极低 | 市场结构不同 |

### 原则 4：学术辩论与反驳
- 识别论文回应的学术辩论
- 模拟持相反立场者的反驳
- 评估论文回应力度

### 原则 5：缺失关键文献
- 因果推断方法论文献
- 批判性观点文献
- 最新文献（近 3 年）

## 方法论检查清单

### 因果识别策略

| 方法 | 必检项 |
|------|--------|
| OLS | 控制变量充分性、遗漏变量偏误 |
| IV | 第一阶段 F>10、排他性约束 |
| DID | 平行趋势、混淆政策排除 |
| RDD | 带宽敏感性、密度操纵检验 |

### 内生性处理（必检）
- [ ] 是否明确识别内生性来源
- [ ] 是否讨论 OLS 估计为何有偏
- [ ] 所选方法如何解决该内生性

### 面板模型（必检 5 项）
- [ ] 聚类标准误（双向聚类）
- [ ] 固定效应设定（个体 FE + 时间 FE）
- [ ] 控制变量质量（避免坏控制变量）
- [ ] 面板内生性处理

> **坏控制变量警示**：DID 研究中控制政策发生后的企业营收、ROA 等属于典型坏控制变量

### 稳健性检验（至少 3 项）
- [ ] 替换被解释变量/核心解释变量
- [ ] 改变样本范围
- [ ] 安慰剂检验

## 评审流程

### 第一阶段：初评
通读摘要后思考：最强版本的论点是什么？最 hostile 的审稿人会攻击哪里？

### 第二阶段：深度评审
每个章节至少读两遍——一遍读内容，一遍读逻辑。

### 第三阶段：输出报告

**正常版**：总体评价 → 分项评述 → 核心问题 → 修改建议 → 结论

**批判版**：执行摘要 → 因果推断审计 → 可推广性审计 → 学术辩论 → 传统维度 → 结论

---

## 深度思考检查

- [ ] 能否用一句话重述核心研究问题？
- [ ] 是否考虑了"这篇文章可能是错的"？
- [ ] 是否存在其他合理的解释？
- [ ] 方法是否最优？有无更好的方法？
- [ ] 批评是否包含"如何改进"的具体建议？
