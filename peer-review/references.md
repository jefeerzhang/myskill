# 参考资料速查

> 同行评议时常用的报告标准、方法检查清单和期刊规范。快速查阅使用。

---

## 常用报告标准（Reporting Standards）

| 标准 | 适用领域 | 核心要求 |
|------|---------|---------|
| **CONSORT** | 随机对照试验 | 随机化、盲法、基线资料、结果完整性 |
| **STROBE** | 观察性研究（社科常用） | 研究设计、样本选择、变量测量、偏倚控制 |
| **PRISMA** | 系统综述/Meta 分析 | 检索策略、筛选流程、文献质量评估 |
| **GB/T 7714** | 中文期刊参考文献 | 格式规范、引用完整 |

**人文社科评审重点关注**：
- STROBE 规范（观察性研究/实证研究）
- 中文期刊投稿指南的具体格式要求
- 国家自然科学基金/社科基金申请书撰写规范

## 人文社科方法论完整检查清单

### 因果识别策略审查

| 方法 | 核心检查要点 | 常见致命问题 |
|------|-------------|--------------|
| **OLS** | 控制变量充分性、函数形式设定 | 遗漏变量偏误、反向因果、函数形式误设 |
| **工具变量（IV）** | 第一阶段 F>10、排他性约束论证 | 弱工具变量、排他性不成立 |
| **双重差分（DID）** | 平行趋势检验、安慰剂检验、混淆政策排除 | 非平行趋势、混淆政策未控制 |
| **断点回归（RDD）** | 带宽敏感性、密度操纵检验 | 带宽选择偏误、样本操纵 |
| **倾向得分匹配（PSM）** | 共同支撑区域、平衡性检验 | 选择偏差、过度拟合 |
| **合成控制法（SCM）** | 预测变量匹配、排列检验 | 稀疏权重、过度拟合 |
| **事件研究法** | 窗口期选择、预期效应检验 | 窗口期过长、预期效应未控制 |
| **交错 DID** | 异质性处理效应处理（Callaway & Sant'Anna / Sun & Abraham） | TWFE 估计有偏 |

---

### 内生性处理专项

#### 工具变量（IV/2SLS）
- [ ] 第一阶段 F 统计量 > 10（弱工具变量检验）
- [ ] 排他性约束（exclusion restriction）是否有充分论证
- [ ] 工具变量是否具有经济含义解释
- [ ] 是否报告了 Sargan/Hansen 过度识别检验
- [ ] 是否进行了 Durbin-Wu-Hausman 内生性检验

#### 双重差分（DID）
- [ ] 平行趋势检验（图示 + 数值检验）
- [ ] 安慰剂检验（placebo treatment）
- [ ] placebo period 检验（提前政策时点）
- [ ] 是否排除了同期混淆政策
- [ ] 是否检验了动态效应（事件研究图）
- [ ] 交错 DID 是否处理了异质性处理效应

#### 断点回归（RDD）
- [ ] 带宽选择是否敏感（不同带宽下结果是否稳健）
- [ ] 是否进行了密度操纵检验（McCrary test）
- [ ] 协变量在断点处是否平衡
- [ ] 是否报告了局部平均处理效应（LATE）
- [ ] 是否有清晰的带宽选择依据（MSE 最优/IK 算法）

#### 倾向得分匹配（PSM）
- [ ] 共同支撑区域是否足够（common support）
- [ ] 平衡性检验是否通过（匹配后标准化偏差<5%）
- [ ] 卡尺选择是否有依据
- [ ] 近邻匹配数量是否合理
- [ ] 是否报告了匹配前后偏差变化

---

### 稳健性检验清单（至少 3 项）

- [ ] **替换被解释变量**：不同度量方式
- [ ] **替换核心解释变量**：不同度量方式
- [ ] **改变样本范围**：剔除特殊年份/行业/地区/极端值
- [ ] **改变聚类层级**：企业/行业/省份/时间聚类
- [ ] **Winsorize 处理**：1%/99% 或 5%/95% 缩尾
- [ ] **更换估计方法**：FE vs RE vs OLS vs GMM
- [ ] **逐步加入控制变量**：检验系数稳定性
- [ ] **安慰剂检验**：随机生成处理组/伪政策时点
- [ ] **Bootstrap 标准误**：小样本稳健性
- [ ] **排除极端值**：剔除前后 1% 或 5%

---

### 机制检验标准

**中介效应**
- [ ] 中介变量是否满足因果逻辑顺序（X→M→Y）
- [ ] 是否排除了替代解释
- [ ] 是否进行了 Sobel 检验或 Bootstrap 检验
- [ ] 中介变量是否可靠测量

**调节效应**
- [ ] 调节变量是否有理论依据
- [ ] 是否报告了简单斜率检验
- [ ] 是否绘制了交互效应图

**异质性分析**
- [ ] 分组依据是否有理论支撑
- [ ] 分组标准是否合理（中位数/均值/自然断点）
- [ ] 是否报告组间系数差异检验（Chow test/Suest）
- [ ] 是否有经济意义解释

---

### 数据审查

**面板数据**
- [ ] 是否说明了面板结构（T 和 N）
- [ ] 是否处理了非平衡面板
- [ ] 固定效应/随机效应选择是否有理论或检验依据（Hausman 检验）
- [ ] 是否控制了时间固定效应
- [ ] 是否控制了个体固定效应

## 面板模型处理检查清单（新增）

> **核心原则**：面板数据的估计质量取决于标准误处理、模型设定和控制变量选择。

### 1. 标准误处理（必检）

| 聚类方式 | 适用场景 | 检查要点 |
|---------|---------|---------|
| **不聚类（OLS 标准误）** | 独立同分布假设成立 | 面板数据中极少适用，需检验 |
| **个体聚类（单向）** | 同一个体不同年份之间存在相关性 | 最常用，允许个体内自相关 |
| **时间聚类（单向）** | 同一年份不同个体之间存在截面相关 | 适用于共同冲击场景 |
| **双向聚类** | 个体内自相关 + 截面相关同时存在 | 最稳健，推荐默认使用 |
| **行业/省份聚类** | 扰动项在更高层级相关 | 聚类到"扰动项相关的最大层级" |

**聚类层级选择原则**：
> "尽量聚类到你认为扰动项相关的最大层级"（Angrist & Pischke, 2009）

**检查清单**：
- [ ] 是否说明了聚类层级的选择理由
- [ ] 是否考虑了双向聚类（个体 + 时间）
- [ ] 如果个体聚类与双向聚类结果差异大，是否讨论了原因
- [ ] 聚类层级数量是否足够（至少 30-50 个聚类单元）
- [ ] 是否报告了聚类标准误与不聚类标准误的对比

**常见错误**：
- 🔴 仅报告 OLS 标准误（忽略面板自相关）
- 🔴 聚类单元过少（如仅 2-3 个省份）
- 🔴 未考虑截面相关性（共同宏观经济冲击）

---

### 2. 模型设定（必检）

| 模型类型 | 适用场景 | 检查要点 |
|---------|---------|---------|
| **混合 OLS** | 个体异质性可忽略 | 需检验个体效应是否存在（F 检验） |
| **固定效应（FE）** | 个体异质性与解释变量相关 | 默认选择，吸收不随时间变化的个体特征 |
| **随机效应（RE）** | 个体异质性与解释变量无关 | 需通过 Hausman 检验 |
| **双向固定效应（TWFE）** | 同时控制个体 + 时间效应 | 实证研究标准配置 |
| **高维固定效应** | 需要控制更多维度 | 行业×年份、省份×年份等交乘 FE |

**固定效应选择检查清单**：
- [ ] 是否说明了固定效应选择的理论依据
- [ ] 是否控制了个体固定效应（吸收不随时间变化的个体特征）
- [ ] 是否控制了时间固定效应（吸收共同时间趋势）
- [ ] 是否需要控制更高层级固定效应（比研究对象层级更高的任意维度，如行业/省份/城市/区域等）
- [ ] 是否需要控制更高层级×年份交乘 FE（吸收更高层级的特异性时间趋势）

**交乘固定效应的应用场景**：
| 交乘 FE | 控制内容 | 典型场景（研究对象：企业） |
|--------|---------|-------------------------|
| 行业×年份 | 行业特异性时间趋势 | 行业政策冲击、技术变革 |
| 省份×年份 | 省份特异性冲击 | 区域政策、地方经济波动 |
| 城市×年份 | 城市特异性冲击 | 城市政策、地方营商环境变化 |
| 区域×年份 | 大区域特异性冲击 | 东部/中部/西部差异化趋势 |

**核心原则**：交乘固定效应的层级选择应**高于研究对象层级**，用于吸收该层级的特异性时间趋势或冲击。

**不同研究对象的更高层级 FE 选择**：
| 研究对象 | 可选更高层级 | 适用场景 |
|---------|-------------|---------|
| 企业 | 行业、省份、城市、集团 | 行业冲击、区域政策、城市效应 |
| 城市 | 省份、区域（东/中/西） | 省级政策、大区域趋势 |
| 省份 | 区域（东/中/西）、全国 | 区域协调政策、全国趋势 |
| 家庭 | 社区、区县、城市 | 社区效应、地方政策 |
| 个人 | 家庭、班级、学校、区县 | 家庭效应、同伴效应、学校效应 |

**常见错误**：
- 🔴 TWFE 中核心解释变量被固定效应完全吸收
- 🔴 过度控制交乘 FE 导致自由度不足
- 🔴 未说明固定效应选择的理由
- 🔴 固定效应层级选择与研究对象层级混淆（如控制企业×年份 FE 吸收核心解释变量变异）

---

### 3. 面板数据的内生性（必检）

**内生性来源**：
| 来源 | 表现 | 解决方案 |
|------|------|---------|
| 遗漏变量 | 不随时间变化的个体特征 | 固定效应模型 |
| 遗漏变量 | 随时间变化的混淆因素 | 加入时变控制变量 |
| 反向因果 | Y 影响 X（如投资影响融资） | 工具变量、滞后解释变量 |
| 测量误差 | 核心变量测度不准确 | 工具变量、替代指标 |
| 样本选择 | 非随机样本进入 | Heckman 选择模型、PSM |

**面板内生性处理检查清单**：
- [ ] 是否明确识别了内生性来源
- [ ] 固定效应是否吸收了不随时间变化的遗漏变量
- [ ] 是否使用了滞后解释变量（缓解反向因果）
- [ ] 是否使用了工具变量（IV-2SLS、GMM）
- [ ] 动态面板是否使用了系统 GMM（Arellano-Bond 检验）
- [ ] 是否讨论了剩余内生性的可能影响

## 控制变量质量检查（新增）

> **核心原则**：控制变量的"好"与"坏"取决于其在因果结构中的位置，而非统计显著性。

### 好控制变量 vs 坏控制变量

| 维度 | 好控制变量 | 坏控制变量 |
|------|-----------|-----------|
| **因果位置** | 处理变量的**前定变量**（先于处理变量确定） | 处理变量的**后定变量**（受处理变量影响） |
| **因果识别** | 消除选择性偏误，逼近随机化实验 | 引入新的选择性偏误，破坏研究设计 |
| **统计推断** | 与处理变量无关，减少噪音、提高精度 | 与处理变量高度相关，增大标准误 |

---

### 坏控制变量的三种类型

#### 类型 1：对撞变量（Collider）

**定义**：同时受到处理变量 D 和影响结果变量 Y 的其他因素 u 影响的变量。

**引入偏误机制**：
- 不控制对撞变量 W 时，D 与 Y 之间仅存在因果效应
- 控制 W 后，D 和 u 通过 W 产生关联，u 成为遗漏变量导致选择性偏误

**典型示例**：
| 研究问题 | 对撞变量 | 偏误来源 |
|---------|---------|---------|
| 大学教育→收入 | 职业选择 | 职业同时受教育和个人能力影响 |
| 数字化→企业绩效 | 企业创新投入 | 创新投入同时受数字化和管理能力影响 |
| ESG 表现→企业价值 | 分析师覆盖 | 分析师覆盖同时受 ESG 和公司透明度影响 |

**识别方法**：
- [ ] 该变量是否可能同时受处理变量和其他因素影响？
- [ ] 控制该变量后，核心系数是否发生显著变化？
- [ ] 是否有理论依据说明该变量是前定变量？

---

#### 类型 2：中介变量（Mediator）

**定义**：位于处理变量→结果变量因果路径上的变量。

**双重负面影响**：
1. **过度控制**：阻断间接效应路径，只估计直接效应而非总效应
2. **引入偏误**：中介变量同时也是对撞变量

**典型示例**：
| 研究问题 | 中介变量 | 后果 |
|---------|---------|------|
| 数字化→企业绩效 | 生产效率 | 控制后只估计直接效应，忽略通过效率提升的间接效应 |
| 政策冲击→投资 | 融资约束 | 控制后阻断政策通过缓解融资约束影响投资的路径 |
| ESG→企业价值 | 声誉提升 | 控制后低估 ESG 的总效应 |

**识别方法**：
- [ ] 该变量是否位于处理变量→结果变量的因果路径上？
- [ ] 控制该变量后，核心系数是否大幅下降或符号反转？
- [ ] 该变量是否在时间上晚于处理变量？

---

#### 类型 3：统计推断方面的坏控制

**定义**：与处理变量高度相关但与误差项不相关的变量。

**影响**：
- 不会导致选择性偏误
- 但会大大降低处理变量的变异性
- 导致标准误显著增加，降低统计功效

**识别方法**：
- [ ] 该变量与核心解释变量的相关系数是否>0.7？
- [ ] 控制该变量后，核心系数的标准误是否大幅增加？
- [ ] VIF 是否>10（多重共线性检验）？

---

#### 类型 4：综合指标的子指标（新增）

**定义**：当被解释变量采用**综合指标**（指数/评分/综合测度）时，该指标体系中的**一级指标、二级指标**均属于坏控制变量。

**核心逻辑**：
- 综合指标 = 一级指标 = 二级指标的**加总/合成**
- 控制子指标 = 控制被解释变量的**组成部分**
- 相当于控制了**结果本身**，导致因果路径被阻断

**典型示例**：

| 研究问题 | 被解释变量（综合指标） | 坏控制变量（子指标） | 后果 |
|---------|---------------------|-------------------|------|
| 数字化→农业现代化 | 农业现代化指数（4 维度） | 生产现代化、产出现代化、生态现代化、安全现代化 | 控制后核心系数被严重低估 |
| ESG→企业价值 | ESG 综合评分 | E 评分、S 评分、G 评分 | 控制后无法估计 ESG 的总效应 |
| 金融发展→经济高质量发展 | 高质量发展指数 | 创新/协调/绿色/开放/共享指数 | 控制后阻断因果路径 |
| 教育投入→学生综合素质 | 综合素质评分 | 学业成绩/体育/艺术/品德评分 | 控制后仅能估计"剩余"效应 |

**数学解释**：
```
假设 Y = w1×Y1 + w2×Y2 + w3×Y3（Y 为综合指标，Y1/Y2/Y3 为子指标）

回归模型：Y = β×X + γ×Y1 + δ×Y2 + ε

问题：控制 Y1、Y2 后，β 估计的是 X 对 Y3 的效应（而非对 Y 的总效应）
      且权重被扭曲：β' = β×w3（而非β）
```

**识别方法**：
- [ ] 被解释变量是否为综合指标（指数/评分/综合测度）？
- [ ] 控制变量是否来自该综合指标的指标体系？
- [ ] 控制变量是否为一级/二级/三级子指标？
- [ ] 控制后核心系数是否大幅下降或变得不显著？

**正确做法**：
| 错误做法 | 正确做法 |
|---------|---------|
| 控制综合指标的子指标 | 不控制任何子指标 |
| 想检验"净效应" | 使用机制检验（中介效应），而非控制变量 |
| 担心"混淆" | 子指标不是混淆变量，是结果的组成部分 |

**DID 研究中的特殊警示**：
```stata
// 错误：被解释变量是农业现代化指数，控制其子指标
reghdfe agr_modern digitization prod_modern output_modern, absorb(id year)
// 问题：prod_modern、output_modern 是 agr_modern 的组成部分

// 正确：仅控制前定控制变量，不控制子指标
reghdfe agr_modern digitization controls_pre, absorb(id year)
```

**常见错误场景**：
- 🔴 被解释变量：企业创新综合指数 → 控制：专利申请数、研发投入、新产品收入
- 🔴 被解释变量：乡村振兴综合评分 → 控制：产业兴旺、生态宜居、乡风文明指数
- 🔴 被解释变量：数字经济指数 → 控制：数字基础设施、数字产业、数字金融指数
- 🔴 被解释变量：高质量发展指数 → 控制：五大发展理念分项指数

### DID 中坏控制变量的特殊问题

**问题背景**：许多 DID 研究在回归模型中添加了企业营收、资产规模、ROA 等**时变控制变量**。

**关键问题**：
- 政策发生后的企业特征必然受到政策影响
- 这些变量属于**后定变量**
- 控制后会阻断政策影响的因果路径
- 可能导致估计偏误方向和大小都不确定

**典型错误示例**：
```stata
// 错误：控制政策发生后的时变变量
reghdfe invest digitization size roa leverage, absorb(id year)

// 正确：仅控制政策前定特征
reghdfe invest digitization size_pre roa_pre, absorb(id year)
```

**DID 控制变量检查清单**：
- [ ] 所有时变控制变量是否为政策前定变量？
- [ ] 如使用政策后变量，是否说明了不会受政策影响？
- [ ] 是否避免控制企业营收、资产规模、ROA 等典型后定变量？
- [ ] 如必须控制，是否使用政策前一年或前几年均值？
- [ ] 是否报告了有无控制变量的结果对比？

---

### "既好且坏"的控制变量（复杂情形）

某些变量可能**既是好控制变量又是坏控制变量**，需要权衡取舍。

**典型案例——户籍变量**：
| 角色 | 影响 | 处理 |
|------|------|------|
| 作为遗漏变量 | 户籍决定教育机会和收入 | 应控制 |
| 作为中介变量 | 教育是改变户籍的途径 | 不应控制 |
| **权衡** | 大多数研究选择控制（遗漏变量偏误更严重） | 两害相权取其轻 |

**处理原则**：
1. 明确主要偏误来源（遗漏变量 vs 过度控制）
2. 报告有无该控制变量的结果对比
3. 在稳健性检验中讨论敏感性

---

### 控制变量选择实践建议

**推荐做法**：
- [ ] 控制变量选择由理论驱动，而非数据驱动
- [ ] 优先选择前定变量（政策前/基期特征）
- [ ] 避免控制因果路径上的变量
- [ ] 警惕共同结果变量（对撞变量）
- [ ] 报告有无控制变量的结果对比
- [ ] 避免过度解读控制变量系数（本身无因果含义）

**DID 研究中的替代方案**：
| 方案 | 操作 | 优点 |
|------|------|------|
| 政策前均值 | 使用政策前 3-5 年均值作为控制 | 避免后定变量问题 |
| 政策前趋势 | 控制被解释变量的政策前趋势 | 吸收个体特异性时间趋势 |
| 不控制时变量 | 仅控制不随时间变化的个体特征 + FE | 避免坏控制变量 |
| 敏感性分析 | 报告有无控制变量的结果对比 | 展示结果稳健性 |

### 坏控制变量识别流程图

```
开始 → 该变量是否在时间上先于处理变量确定？
         ↓
       是 → 好控制变量候选 → 检查是否与处理变量高度相关？
         ↓                       ↓
       否                      是 → 统计推断坏控制
         ↓                       ↓
    后定变量候选              否 → 好控制变量
         ↓
    是否位于因果路径上？
         ↓
       是 → 中介变量（坏控制）
         ↓
       否
         ↓
    是否同时受 D 和 u 影响？
         ↓
       是 → 对撞变量（坏控制）
         ↓
       否 → 可能是好控制变量
```

## 常见方法统计问题索引

| 问题类别 | 具体表现 | 检查要点 |
|---------|---------|---------|
| **样本量不足** | 统计功效低、结论不可靠 | Power analysis、效应量估算 |
| **多重比较问题** | p 值膨胀、假阳性 | Bonferroni/FDR 校正 |
| **选择性报告** | 只报显著结果 | 是否包含阴性结果 |
| **过度拟合** | 模型复杂度高、泛化差 | 交叉验证、样本外检验 |
| **遗漏变量** | OLS 有偏 | 固定效应、IV、PSM |
| **反向因果** | 因果方向不明确 | 工具变量、DID、RDD |
| **异方差/序列相关** | 标准误有误 | 稳健标准误、聚类标准误 |

### 常见退稿原因（中文期刊）
