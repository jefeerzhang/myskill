# 同行评议 Skill（Peer Review Skill）

> 🎓 学术论文同行评审专家 | 适配人文社科领域（经济学、金融学、管理学、发展经济学）

## 📋 功能概述

本 skill 是一个**学术论文同行评审专家**，提供系统化的论文评审框架，涵盖：

- ✅ 7阶段评审流程（初评→逐节→方法→可重复性→图表→伦理→写作）
- ✅ **人文社科方法论检查清单**（因果推断、内生性处理、稳健性检验）
- ✅ **中文期刊审稿规范**（国自然评审、学位论文评审）
- ✅ **审稿意见回复模板**（中英文双版本）
- ✅ 强制深度思考模式（9项检查清单）

| 场景 | 说明 |

## 📝 核心内容

### 1. 人文社科方法论检查清单

- **因果识别策略**：OLS/IV/DID/RDD/PSM/SCM/事件研究
- **内生性处理**：IV（F>10）、DID（平行趋势）、RDD（带宽敏感性）
- **稳健性检验**：10项标准检验清单
- **机制检验**：中介效应、调节效应、异质性分析
- **数据审查**：面板数据、调查数据、常用数据库

### 2. 中文期刊审稿规范

- 基本格式检查（摘要/关键词/参考文献）
- 常见退稿原因分析
- 国家自然科学基金评审要点
- 学位论文评审标准（硕士/博士）

### 3. 审稿意见回复模板

- 中文回复信模板
- 英文回复信模板
- 回复策略指南
- 常见审稿意见回复模板（内生性/样本量/机制检验/稳健性）
