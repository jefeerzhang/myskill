# 社科基金题目评审指南

## 一、触发条件

当用户请求以下类型时触发本技能：
- `题目评审`
- `评审题目`
- `看看这个题目`
- `申报题目`
- `这个题目怎么样`

---

## 二、硬性约束（一票否决项）

评审前首先检查以下约束，任一不满足即判定为不合格：

| 约束项 | 要求 | 不合格示例 |
|--------|------|------------|
| **题目长度** | 绝对不超过30个字（不含标点） | `超过30个字的冗长题目...研究` |
| **副标题** | 禁止使用 | `主标题——基于XX的实证研究` |
| **核心概念数量** | 3-4个（推荐3个） | 超过4个独立概念 |
| **概念清晰度** | 必须能界定清楚，无歧义 | 含混不清、内涵不明的概念 |
| **范围控制** | 不能范围太大 | `中国发展研究`、`社会治理创新` |
| **主题明确性** | 主题必须清晰 | 看完不知研究什么 |
| **内容清晰度** | 研究内容可预期 | 无法判断具体研究什么 |

---

## 三、评审四维框架

### 维度1：问题意识凸显（25%）

**定义**：运用现代科学知识与方法，从纷繁复杂的社会现象中提炼影响事物本身发展规律的核心问题，或分析其产生原因，或阐述其后果，或从理论高度提炼解释框架，或从实践中寻找对策，或对其未来走势作出科学预判的不断追问之过程。

**评审要点**：
- [ ] 题目是否体现对现实问题的敏锐捕捉？
- [ ] 是否展现因果追问（原因/后果）？
- [ ] 是否有理论解释框架的提炼意图？
- [ ] 是否有实践对策导向？
- [ ] 是否有预判或前瞻性？

**优秀标准**：具有强烈的问题导向，能引发评审专家的探究兴趣。

---

### 维度2：研究对象明确

**定义**：核心概念要能够界定清楚，不要含混；要考虑拿下来后如何做好研究（申报时也如此，否则就设计不好后面的研究思路与研究框架）。

**评审要点**：
- [ ] 核心概念是否能清晰界定？
- [ ] 概念内涵是否明确无歧义？
- [ ] 研究对象是否具体可识别？
- [ ] 后续研究设计是否可预期？
- [ ] 研究边界是否清晰？

**优秀标准**：看一眼题目就知道研究什么，核心概念可定义、可操作。

---

### 维度3：研究方法可行

**定义**：在讲清课题研究的具体方法时，可以从技术路线图视角对整个课题的内容设计进行谋篇布局，从而将研究内容、所涉理论、所用方法、所含步骤有机统一在一张图中，既形象直观，又生动丰富。

**评审要点**：
- [ ] 题目是否暗示了可行的研究方法？
- [ ] 研究路径是否可从题目中预期？
- [ ] 理论视角是否隐含其中？
- [ ] 研究步骤是否可推演？
- [ ] 方法与研究对象是否匹配？

**优秀标准**：从题目可预见研究的技术路线图，方法与研究问题高度匹配。

---

### 维度4：研究空间打开

**定义**：好的选题，既要有视觉冲击力，体现强烈的问题意识，又要有深层的价值关怀，关注制度建构，体现古今中外相结合之用。

**评审要点**：
- [ ] 题目是否有视觉冲击力？
- [ ] 是否有深层价值关怀？
- [ ] 是否关注制度建构？
- [ ] 是否体现理论对话空间？
- [ ] 是否具有实践应用价值？

**优秀标准**：题目让人眼前一亮，既有学术深度又有现实关怀。

## 四、评审流程

当用户请求评审题目时，按以下步骤执行：

### 步骤1：硬性约束检查（一票否决）
1. 检查题目长度（≤30字）
2. 检查是否有副标题
3. 检查核心概念数量（≤3个）
4. 检查概念清晰度
5. 检查范围是否过大
6. 检查主题和内容清晰度

**如任一约束不满足**：
- 直接判定为"不合格"
- 指出具体违规项
- 给出修改方向建议
- 结束评审

### 步骤2：四维评审（约束全满足时）
对题目进行四个维度的深度评审，给出简要评语，**不计算总分，不给等级**。

---

## 五、输出格式

### 输出要求
1. **总体评价**：是否合格/优秀（一句话总结）
2. **四维评分**：问题意识/研究对象/研究方法/研究空间（每项分数+等级+简要评语）

### 输出模板

```markdown
## 题目评审报告

**原题**：XXX

---

### 一、硬性约束检查

| 约束项 | 结果 | 说明 |
|--------|------|------|
| 题目长度（≤30字） | ✅/❌ | 当前X字 |
| 无副标题 | ✅/❌ | - |
| 核心概念3-4个（推荐3个） | ✅/❌ | 识别到X个概念：... |
| 概念清晰 | ✅/❌ | - |
| 范围适中 | ✅/❌ | - |
| 主题明确 | ✅/❌ | - |
| 内容清晰 | ✅/❌ | - |

**硬性约束结论**：✅ 全部满足 / ❌ 存在违规项

---

### 二、总体评价

[一句话评价：如"优秀题目，问题意识强烈，研究对象明确，建议直接申报"]

---

### 三、四维评审

| 维度 | 评语 |
|------|------|
| 问题意识凸显 | [简要评语] |
| 研究对象明确 | [简要评语] |
| 研究方法可行 | [简要评语] |
| 研究空间打开 | [简要评语] |

---

[如不合格，给出修改方向建议]
```

## 六、优秀题目特征总结

### 好题目的标准
1. **字数**：20-28字为宜
2. **结构**：研究对象 + 核心问题 + 方法/视角
3. **概念**：2-3个核心概念，清晰可界定
4. **问题**：有明确的问题导向
5. **方法**：隐含可行的研究路径
6. **价值**：有理论或实践价值

### 常见不合格类型
| 类型 | 特征 | 示例 |
|------|------|------|
| 范围过大 | 题目宏大，无法操作 | `中国农村发展研究` |
| 概念含混 | 核心概念界定不清 | `基于复杂系统理论的研究` |
| 概念过多 | 超过3个核心概念 | `A+B+C+D的机制与路径研究` |
| 方法不明 | 看不出如何研究 | `XX问题研究` |
| 副标题泛滥 | 使用破折号副标题 | `XX研究——基于YY的视角` |
