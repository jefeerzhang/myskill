# 开题报告评审专家 - 快速上手

## 3分钟上手

1. **上传开题报告**：直接粘贴文本，或上传 PDF 文件
2. **选择学位层级**：本科 / 硕士 / 博士
3. **开始评审**：技能会自动进行五维评估 + 致命伤预警

## 学位层级选择

| 层级 | 评审重点 | 创新性要求 |
|------|---------|-----------|
| **本科** | 规范性 + 逻辑闭环 | 允许应用成熟理论 |
| **硕士** | 文献对话 + 方法严密 | 理论视角补充/机制深挖 |
| **博士** | 理论贡献 + 因果识别 | 显著边际贡献 |

## 可选：RAG 文献验证

如需启用 RAG 文献验证，在技能目录下创建 `.rag_config.json`：

```json
{
  "base_url": "http://localhost:3001",
  "api_key": "your-api-key",
  "workspace_slug": "your-workspace-id"
}
```

## 可选：PDF 提取

如需 PDF 自动提取功能，安装 MinerU：

```bash
npm install -g mineru-open-api
```

## 评审报告

评审完成后会自动保存到 `./开题报告评审/` 目录。
