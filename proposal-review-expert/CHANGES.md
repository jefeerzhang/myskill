## 使用方法

### 启用 RAG 评审

```
用户：评审这份硕士开题，启用 RAG
[粘贴开题报告内容]
```

### 配置 AnythingLLM

1. 确保 AnythingLLM 已启动并运行
2. 确认工作区已有相关文献
3. 编辑 `.rag_config.json` 配置正确的 API Key 和 Workspace Slug

### 查询示例

```python
# 手动测试 RAG 查询
cd SKILLs/proposal-review-expert
python rag_query.py
```

编辑 `.rag_config.json` 文件，填入你的 AnythingLLM 配置。

## 注意事项

1. **API 调用次数**：每次评审最多 3 次 RAG 查询
2. **超时设置**：默认 180 秒，可在代码中调整
3. **错误处理**：如 RAG 查询失败，评审仍可继续（降级为纯文本评审）
4. **文献库质量**：RAG 效果取决于 AnythingLLM 中已有文献的质量和数量
