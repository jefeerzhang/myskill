## 🚀 快速安装（3 分钟）

### 前置要求

1. **OpenClaw / LobsterAI** - 已安装并运行
2. **Node.js** - v16 或更高版本

### 步骤 1：安装 Node.js（如未安装）

**Windows**：
- 下载安装：https://nodejs.org/zh-cn/
- 选择 LTS 版本（推荐 v20.x）

**macOS**：
```bash
brew install node
# 或访问 https://nodejs.org/zh-cn/
```

**Linux**：
```bash
curl -fsSL https://deb.nodesource.com/setup_lts.x | sudo -E bash -
sudo apt-get install -y nodejs
```

**验证安装**：
```bash
node --version  # 应显示 v16.x 或更高
npm --version   # 应显示 8.x 或更高
```

---

### 步骤 2：安装 MinerU CLI

**所有平台通用**：
```bash
npm install -g mineru-open-api
```

**macOS/Linux 如遇到权限问题**：
```bash
npm install -g mineru-open-api --prefix ~/.local
```

**验证安装**：
```bash
mineru-open-api version
```

看到类似输出即表示成功：
```
mineru-open-api version v0.2.1
```

---

### 步骤 3：安装技能

**macOS / Linux**：
```bash
# 方法 A：使用 cp 命令
cp -r academic-literature-guide-v2 ~/Library/Application\ Support/LobsterAI/SKILLs/

# 方法 B：手动复制
# 将 academic-literature-guide-v2 文件夹复制到：
# ~/Library/Application Support/LobsterAI/SKILLs/
```

**Windows**：
```powershell
# 方法 A：使用 PowerShell
Copy-Item -Path "academic-literature-guide-v2" -Destination "$env:APPDATA\LobsterAI\SKILLs\" -Recurse

# 方法 B：手动复制
# 将 academic-literature-guide-v2 文件夹复制到：
# %APPDATA%\LobsterAI\SKILLs\
```

**重启 LobsterAI**（如已运行）

---

### 步骤 4：验证安装

在 LobsterAI 中发送以下消息：
```
文献导读技能
```

如果技能已正确安装，应该会看到技能的触发提示。

## 🔑 API Token 配置（可选但推荐）

### 为什么要配置 Token？

| 模式 | 适用场景 | 是否需要 Token |
|------|---------|---------------|
| **flash-extract** | 小文件（<10MB, <20 页） | ❌ 无需 Token |
| **extract** | 大文件或复杂排版 | ✅ 需要 Token |

### 获取 Token

1. 访问 https://mineru.net/apiManage/token
2. 注册免费账号
3. 登录后点击「创建 Token」
4. 复制生成的 Token（类似：`sk-xxxxxxxxxxxxxxxx`）

### 配置 Token

```bash
mineru-open-api auth
```

按提示粘贴 Token 即可。

**验证 Token**：
```bash
mineru-open-api auth --verify
```

## 📖 使用方法

### 基本用法

**本地 PDF 文件**：
```
帮我解读这篇 PDF：/path/to/paper.pdf
```

**工作目录中的文件**：
```
解读一下"烧钱行为"那篇 PDF
```

**URL 链接**：
```
帮我解读这个链接的论文：https://arxiv.org/pdf/2509.22186

**DOI 或标题**：
```
帮我生成 10.1080/xxxx 这篇论文的导读

4. **🔴 批判层** - 培养科学怀疑精神

报告自动保存到 `./文献导读/` 目录。

## ❓ 常见问题

### Q1: 安装时提示权限不足

**macOS/Linux**：
```bash
npm install -g mineru-open-api --prefix ~/.local
```

**Windows**：
- 以管理员身份运行 PowerShell
- 或修改 npm 全局目录权限

### Q2: flash-extract 和 extract 有什么区别？

| 特性 | flash-extract | extract |
|------|--------------|---------|
| 速度 | 快（5-10 秒/页） | 中等（10-30 秒/页） |
| 精度 | 标准 | 高精度 |
| Token | 无需 | 需要 |
| 适用 | 普通 PDF | 复杂排版/表格/公式 |

### Q3: 提取失败怎么办？

1. 检查文件是否可以正常打开（无加密/损坏）
2. 检查网络连接（需要访问 MinerU API）
3. 检查 Token 是否有效：`mineru-open-api auth --verify`
4. 尝试使用另一种模式（flash vs extract）

### Q4: 公式显示异常？

确保你的 Markdown 阅读器支持 LaTeX 渲染：
- VS Code + Markdown All in One 插件 ✅
- Typora ✅
- Obsidian ✅
- 记事本 ❌
